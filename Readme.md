# Project README - Problem Overview

This repository contains solutions and scripts for three distinct problems. Below is an overview of each problem and the topic it addresses.

## Problem 1: Web Scraping OLX for Car Covers
**Topic:** Python web scraping to extract and display search results from OLX website  
- Developed a Python program to scrape OLX car cover listings using the provided URL.  
- Extracted key parameters: title of the ad, description, and price (excluding images).  
- Results are formatted and printed as a table for clear visualization.  
- The code is hosted on GitHub for review and reuse.

## Problem 2: SQL Queries on Rfam Database
**Topic:** Querying biological taxonomy, sequence data, and family information in a public RNA families database  
- Tasks include counting tiger types in taxonomy, retrieving Sumatran tiger NCBI ID, analyzing rice DNA sequences, and paginating family data.  
- Implemented using SQL queries to extract meaningful biological insights.

## Problem 3: Shell Script to Extract Mutual Fund Data
**Topic:** Unix shell scripting for data extraction from a large text file  
- Script downloads NAV data from AMFI website.  
- Extracts Scheme Name and Net Asset Value fields and saves them as a TSV file.  
- Includes considerations about storing tabular data in JSON format for extensibility and integration.

This README provides a snapshot of the problem domain and script functionality for quick reference. Detailed scripts and queries accompany each problem folder or section in this repository.
